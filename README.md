# フランスごクイズ 🇫🇷 (保護者向けメモ)

## ゲームコンセプト「ことばあつめの だいぼうけん」
子供は気球に乗った「ことばあつめの ぼうけんか」。単語に正解すると⭐(おぼえた)が増えて気球が進む。
マイルストーン: 各レベル100%でバッジ獲得 — 🥐A1 → 🗼A2 → 🏰B1 → ⛰️B2 → 🎨C1 → 👑C2、2周目(200%)クリアで✨金色化。
間違えた単語は📦ふくしゅうボックスに「にげこむ」→ 正解して捕まえ直す、という物語仕立て。
初回プロフィール選択時に絵入りチュートリアル2ページが自動表示され、ホームの「▶️ おはなしをみる」でいつでも再表示可能。

### チュートリアルの絵の差し替え
`img/tutorial1.png`(1ページ目)、`img/tutorial2.png`(2ページ目)を置くと自動でその画像が表示されます(jpgの場合はpngに変換するか、index.html内のファイル名を変更)。
ファイルがない場合は内蔵のSVGイラストが表示されます。

## 起動方法
`index.html` をダブルクリックしてブラウザで開くだけ。ネット接続不要・完全ローカル動作。
ChromeまたはSafari推奨。

## 機能
- **プロフィール4人まで**: 最初の画面で選択/追加。記録は人ごとに別管理
- **クイズ2方向**: 絵+日本語→フランス語 / フランス語→絵+日本語(ミックスも可)。1回10問
- **答え合わせ**: 大きな⭕❌+効果音。他の選択肢の意味・読み方も同時に表示され、外れの単語からも学べる
- **発音**: 🔊ボタンでフランス語音声(名詞は le/la 込みで読み上げ)
- **男性/女性名詞**: 名詞は常に冠詞付きで出題。🔵le / 🔴la のバッジ表示
- **ふくしゅうボックス**: 間違えた単語が自動で入る。クイズ(通常・復習どちらでも)で正解すると自動で外れる。「おぼえた!」ボタンで手動でも外せる
- **きろく**: 日毎の回答数/正解数グラフ(14日分)、A1〜C2の到達度バー、全員の比較表
- **バッジシステム**: 各レベル10%到達ごとにバッジ1個(100%超の周回中も継続)。10問セット終了時に新規到達があると全画面のバッジゲット演出(複数同時はタップで順番に表示)。きろく画面に「バッジコレクション」(レベル×10枠+周回分は✨)、ホームに合計数、結果画面に「つぎのバッジまで あと◯もん!」表示
- **到達度の仕組み**: 1回正解で「おぼえた」。レベル内の全単語を覚えると2周目開始(バーはリセット、%は101%→200%と累積)。周回は各単語の最低正解回数で判定。ホーム画面にはA1/A2/B1の3本バーを常時表示
- **設定**: 効果音ON/OFF、カタカナよみがなON/OFF
- **出題モード** (ホーム下部の2ボタンで切替): 「🎯おぼえてないことばだけ」(初期設定・現在の周でまだ覚えていない単語だけ出題、%がどんどん進む) / 「🎲ぜんぶからランダム」(全単語から完全ランダム)。未習が10語未満なら覚えた単語で自動的に埋める

## データ保存について
- 記録はブラウザのlocalStorageに保存。**同じブラウザで開けば残ります**
- ブラウザを変える/PCを変える場合は、ホーム画面の「💾ほぞん」でJSON書き出し→「📂よみこみ」で復元
- ファイルの置き場所を移動するとブラウザによっては記録が別扱いになるため、移動時は先に💾ほぞん推奨

## 発音について
- アプリはブラウザ内のフランス語音声だけを選んで使います(英語・日本語の声にはフォールバックしません)
- 自動選択の標準の声は **Audrey (Premium/Enhanced)**(インストール済みなら常に最優先、なければ他の女性のフランス語音声を優先)。別の声にしたい場合はプルダウンから選択
- **声が気に入らない場合**: ホームのプルダウン「🗣️ こえを えらぶ」で端末内の全フランス語音声を試聴して固定できます(選ぶと bonjour, merci と試し読み)。★付きが高音質版
- **重要**: macOS標準で最初から入っているのは「コンパクト版」というロボット風の低音質声だけのことが多いです(どもって聞こえる)。きれいな声にするには:
  システム設定 → アクセシビリティ → 読み上げコンテンツ → システムの声 → 管理 → フランス語 で
  **Audrey (プレミアム)** や **Amélie (拡張)** をダウンロード → ブラウザ再起動 → アプリのプルダウンで選択
- Chromeならネット接続時「Google français」(女性・高音質)も選べます
- いま使われている声はホーム画面の下部に「🗣️ フランスごの こえ: ○○」と表示されます
- 「⚠️ みつからない」と出る場合、そのブラウザにフランス語音声がありません:
  - **macOS**: システム設定 → アクセシビリティ → 読み上げコンテンツ → システムの声 → 管理 で フランス語(Thomas, Amélie など)を追加 → ブラウザ再起動
  - **Chrome** は「Google français」というネット経由の声を使えることもあります(オフライン時は不可)
- 発音が英語風に聞こえた場合は上記の声が未インストールだった可能性が高いです

## 単語の追加方法
`words.js` に1行追加するだけ。例:
```js
{fr:"fantôme",art:"le",g:"m",ja:"おばけ",kana:"ル ファントム",e:"👻",lv:"A1",cat:"あそび",pos:"n"},
```
- 名詞(pos:"n")は `art`(le/la/l'/les)と `g`(m/f)必須。母音・無音h始まりは `l'`
- `ja` はひらがな・カタカナのみ(子供が読めるように)
- `kana` は冠詞込みの読み(例: l'eau → ロー)
- B2以降を追加する場合は `lv:"B2"` 等にすれば到達度バーが自動で有効になります
